# Interview Preparation

## Purpose
Common Python interview questions, patterns, and behavioral prep notes.

## Common Python Questions
1. What's the difference between a list and a tuple?
2. Explain mutable vs immutable types with examples.
3. What is the GIL and how does it affect multithreading?
4. Difference between `@staticmethod`, `@classmethod`, and instance methods.
5. How does Python's garbage collection work?
6. Explain shallow copy vs deep copy.
7. What are decorators and how do they work under the hood?
8. Difference between `is` and `==`.
9. What are generators and why use them over lists?
10. Explain Python's MRO (Method Resolution Order) and `super()`.

## Coding Patterns to Practice
- Two pointers / sliding window
- Hash map frequency counting
- Recursion & backtracking
- BFS/DFS on graphs/trees
- Dynamic programming basics

## Behavioral Prep
- Have 2-3 STAR-format stories ready (Situation, Task, Action, Result) about real projects.
- Be ready to explain *why* you made specific design decisions in your own projects.

See `dsa-python/` for implementations of the data structures referenced above.
