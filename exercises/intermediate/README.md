# Intermediate Exercises

Covers OOP, file handling, exceptions, iterators/generators, decorators, and functional tools.

## Problems

1. **Bank Account Class** — Model deposit/withdraw with custom exceptions for invalid operations.
2. **CSV Report Generator** — Read a CSV, compute aggregates, write a summary file.
3. **Retry Decorator** — Write a `@retry(times=3)` decorator that retries a failing function.
4. **Custom Range Iterator** — Implement a class-based iterator mimicking `range()`.
5. **Memoization Decorator** — Cache expensive function results using a decorator.

Try solving without external libraries first — these exercises are meant to build core fluency.
