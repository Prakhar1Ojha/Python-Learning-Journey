# Beginner Exercises

Practice problems covering variables, strings, loops, lists, dicts, and basic functions.

## Problems

1. **FizzBuzz** — Print numbers 1–100; multiples of 3 print "Fizz", multiples of 5 print "Buzz", multiples of both print "FizzBuzz".
2. **Palindrome Checker** — Given a string, check if it's a palindrome (ignore case/spaces).
3. **Word Counter** — Count word frequency in a given sentence.
4. **List Deduplication** — Remove duplicates from a list while preserving order.
5. **Simple Calculator** — Take two numbers and an operator as input, return the result.

Solutions go in `solutions/` (create as you solve them) — try the problem yourself first before checking any reference solution.
