# Advanced Exercises

Covers concurrency, async, web/APIs, and small systems-design problems.

## Problems

1. **Multithreaded Downloader** — Download multiple URLs concurrently using `threading` or `concurrent.futures`.
2. **Async Web Scraper** — Use `asyncio` + `aiohttp` to fetch multiple pages concurrently.
3. **Rate Limiter** — Implement a token-bucket rate limiter as a decorator.
4. **LRU Cache from Scratch** — Implement an LRU cache without using `functools.lru_cache`.
5. **Mini REST API** — Build a small CRUD API with Flask or FastAPI backed by SQLite.

These are meant to be portfolio-worthy — write tests for each.
