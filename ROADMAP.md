# 🛣 Learning Roadmap

A topic-by-topic path from absolute basics to advanced, real-world Python. Check items off as you complete them (swap `[ ]` → `[x]`).

## Stage 1 — Foundations
- [ ] Variables & Data Types
- [ ] Operators
- [ ] Strings
- [ ] Lists
- [ ] Tuples
- [ ] Sets
- [ ] Dictionaries
- [ ] Loops (`for`, `while`)
- [ ] Functions
- [ ] Modules & Packages
- [ ] Virtual Environments

## Stage 2 — Intermediate Core
- [ ] File Handling
- [ ] Exception Handling
- [ ] Object Oriented Programming (classes, inheritance, polymorphism, encapsulation)
- [ ] Iterators
- [ ] Generators
- [ ] Decorators
- [ ] Lambda, Map, Filter, Reduce

- [ ] Logging

- [ ] Threading
- [ ] Multiprocessing

## Stage 5 — Web & Automation

## Stage 6 — Web Frameworks
- [ ] Flask (routing, templates, REST API)
- [ ] Django (models, views, admin)
- [ ] FastAPI (async endpoints, Pydantic, docs)

## Stage 7 — Data & ML
- [ ] NumPy
- [ ] Pandas
- [ ] Matplotlib / data visualization

## Stage 8 — Professional Skills
- [ ] Testing (`unittest`, `pytest`)
- [ ] Packaging & Deployment
- [ ] Interview Preparation (DSA in Python + behavioral)

---

**How to use this roadmap:**
1. Pick the next unchecked topic.
2. Read/write notes in `notes/<topic>.md`.
3. Write examples in the matching topic folder's `examples/`.
4. Solve 3–5 problems in `exercises/<level>/`.
5. Check it off here and update the Progress Tracker in the main README.
