# Weekly / Monthly Learning Plan

## Weekly Goals Template
```
Week of: ____
- [ ] Topic 1:
- [ ] Topic 2:
- [ ] 1 project or exercise set completed
- [ ] Notes written in notes/
```

## Monthly Goals Template
```
Month: ____
- [ ] Roadmap stage completed: ____
- [ ] Projects shipped: ____
- [ ] Interview questions reviewed: ____ count
```

## Suggested Pace (example)
| Week | Focus |
|---|---|
| 1 | Basics: variables, types, strings, loops |
| 2 | Data structures: lists, dicts, sets, tuples |
| 3 | Functions, modules, file/exception handling |
| 4 | OOP fundamentals |
| 5 | Iterators, generators, decorators |
| 6 | Build a mini project using everything so far |

Adjust freely — this is a guide, not a contract.
