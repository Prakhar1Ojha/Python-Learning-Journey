# Exception Handling

## Purpose
This folder covers **Exception Handling** as part of the Python Learning Journey roadmap.

## Concepts
try/except/finally, custom exceptions, raising errors

## Structure
- `examples/` - runnable, focused example scripts, one concept per file
- Exercises related to this topic also live in `../exercises/`

## Best Practices
- Keep each example minimal and focused on one idea
- Use type hints and docstrings
- Test your understanding by modifying the examples, not just reading them

## Exercises
1. Re-implement one example from scratch without looking
2. Combine this topic with one previous topic in a small script
3. Add an edge case the example does not cover and handle it
