# API Projects

## Status
🚧 Planned / not yet started.

## Idea
Projects that consume or build REST APIs — weather, jokes, currency conversion, or a custom Flask/FastAPI API.

## Suggested approach
1. Sketch the core feature in plain functions first
2. Add a CLI or simple UI
3. Add error handling and a README usage example
4. (Optional) add tests in a `tests/` folder
